# Дизайн-бриф — HookahBook MVP

## Общие принципы

- **Mobile-first** — 80%+ гостей будут заходить с телефона
- **Тёмная тема** — кальянные заведения = приглушённый свет, тёмный интерфейс органичнее
- **Минимум шагов** — от открытия сайта до бронирования максимум 4 тапа
- **Язык**: русский (MVP для РФ рынка)

## UX-флоу: Конструктор зала (владелец, админ-панель)

```
┌───────────────────────────────────────────────────────────┐
│  АДМИН-ПАНЕЛЬ              Hookah Lounge        [Выйти]  │
├──────────┬────────────────────────────────────────────────┤
│          │                                                │
│  Зал  ←  │  КОНСТРУКТОР ЗАЛА                              │
│  Табаки  │                                                │
│  Брони   │  Палитра:                                      │
│  Заказы  │  [ 🟫 Стол (кв.) ] [ ⚫ Стол (кр.) ] [ ▬ Стена ] │
│          │                                                │
│          │  ┌──────────────────────────────────────────┐  │
│          │  │              CANVAS                      │  │
│          │  │                                          │  │
│          │  │    ┌─────┐         ╭───╮                 │  │
│          │  │    │  1  │         │ 2 │                 │  │
│          │  │    │ 4м  │         │ 6м│                 │  │
│          │  │    └─────┘         ╰───╯                 │  │
│          │  │                                          │  │
│          │  │    ████████████████████████               │  │
│          │  │                                          │  │
│          │  │    ┌─────┐                               │  │
│          │  │    │  3  │                               │  │
│          │  │    │ 2м  │  ← drag & drop               │  │
│          │  │    └─────┘                               │  │
│          │  │                                          │  │
│          │  └──────────────────────────────────────────┘  │
│          │                                                │
│          │  Настройки стола (при выборе):                  │
│          │  Номер: [ 3 ]  Вместимость: [ 2 ]              │
│          │                                                │
│          │  [ 💾 Сохранить ]  [ ↩ Отменить ]              │
└──────────┴────────────────────────────────────────────────┘
```

**Элементы конструктора:**
- **Палитра** (вверху): прямоугольный стол, круглый стол, стена — drag на canvas
- **Canvas**: сетка для выравнивания, масштабирование (pinch-zoom на мобильных)
- **Выделение объекта**: клик → синяя рамка + ручки изменения размера
- **Popup/панель настроек**: номер стола, вместимость (появляется при выборе стола)
- **Стены**: рисуются как линии/прямоугольники, snap to grid
- **Сохранение**: весь план сериализуется в JSON, отправляется на backend

## Адаптивность

| Элемент | Mobile (< 768px) | Desktop (> 1024px) |
|---------|------------------|-------------------|
| Конструктор зала (админ) | Базовый, pinch-zoom | Полный, drag-and-drop |
| Просмотр зала (гость) | Горизонтальный скролл + zoom | Полный план |
| Каталог табаков | Карточки в 1 колонку | Таблица |
| Дашборд бронирований | Карточки | Таблица + план зала |
| Бронирование | Пошаговый wizard | Все шаги на одной странице |

---

## Шрифты и компоненты

- **Шрифт**: Inter (бесплатный, отличная читаемость)
- **UI-библиотека**: Shadcn/UI (React, легковесная, кастомизируемая, тёмная тема из коробки)
- **Иконки**: Lucide (совместимы с Shadcn)
- **Тосты/уведомления**: Sonner (лёгкая, красивая)
