# HookahBook — Project Rules

## Workflow

- После каждой итерации разработки обязательно проводи тесты в превью-режиме (preview_screenshot, preview_eval, preview_click, curl к API) — проверяй и фронтенд, и бэкенд.
- После каждой итерации запускай code review через `/everything-claude-code:code-review` для ревью всех изменённых файлов.
